CI/CD

Azure Pipeline

This lesson covers CI/CD pipelines for Azure and GitHub, including Terraform infrastructure deployment and application delivery.

Azure DevOps Pipelines

  • Terraform pipeline for infra deployment: use Azure DevOps YAML pipeline stages such as plan and apply.
  • State management: store Terraform state in Azure Storage or Terraform Cloud.
  • Authentication: use an Azure service connection with managed identity, workload identity federation, or service principal.
  • Application pipeline: build, test, publish artifacts, and deploy to App Service, AKS, or Web App for Containers.

GitHub Actions Pipelines

  • Create a workflow like .github/workflows/terraform.yml.
  • Use actions/checkout, hashicorp/setup-terraform, and azure/login.
  • Run terraform init, terraform plan, and terraform apply.
  • Use repository secrets or federated credentials for Azure authentication.

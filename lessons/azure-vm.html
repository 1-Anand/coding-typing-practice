Azure Linux VM

This lesson explains how to create and manage a Linux virtual machine in Azure.

Key topics

  • Choose Ubuntu, CentOS, or another Linux distribution.
  • Use Azure Portal, CLI, Terraform, ARM, or Bicep to provision the VM.
  • Open SSH only from trusted source addresses and configure the NIC and NSG.
  • Install packages after deployment and keep the host patched.

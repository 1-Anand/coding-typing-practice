AKS Basic Commands

Practice common Azure Kubernetes Service commands for creating clusters, connecting with kubeconfig, managing node pools, and checking workloads.

Practice files

  • aks-create-and-connect.sh: login, create, credentials, and node checks.
  • aks-standard-cluster.sh: Standard cluster and node pool commands.
  • kubectl-daily-use.sh: namespaces, apply, logs, describe, and exec.
  • aks-operate-and-monitor.sh: scale, rollout, metrics, monitoring, stop, and start.

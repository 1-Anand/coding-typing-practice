Ingress

AGIC YAML

This lesson introduces YAML configuration for Azure Application Gateway Ingress Controller (AGIC) in Kubernetes.

AGIC basics

  • AGIC integrates Azure Application Gateway with AKS ingress routing.
  • You define ingress resources and annotations for Application Gateway behavior.

Common YAML elements

  • apiVersion: networking.k8s.io/v1
  • kind: Ingress
  • metadata.annotations for settings such as appgw.ingress.kubernetes.io/ssl-redirect.
  • spec.rules for host and path routing.

Infrastructure as code

Azure Terraform

This lesson introduces a small Terraform workflow for creating Azure resources from version-controlled code.

Practice workflow

  • Run terraform fmt to keep files consistent.
  • Run terraform validate before opening a pull request.
  • Use terraform plan to review changes before apply.
  • Use a service connection or federated credentials for CI/CD authentication.
  • Keep state in a remote backend for team practice projects.

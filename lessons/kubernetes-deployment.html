Containers

Kubernetes Deployment

This lesson covers deploying applications to Kubernetes using YAML manifests.

Deployment basics

  • Define a Deployment with replicas, container image, and labels.
  • Use a Service to expose pods internally or externally.
  • Use kubectl apply -f to deploy and kubectl rollout status to monitor.

Best practices

  • Set resource requests and limits.
  • Define readiness and liveness probes.
  • Keep deployments declarative and version-controlled.
